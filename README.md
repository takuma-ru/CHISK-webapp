## Description
**現在開発中**<br>
**個人向けタスク管理Webアプリ**<br>
PWA・TWA化をしているので、ネイティブアプリに並ぶ操作性があります。<br>
タスクをこなすと**地球**にたくさんの生き物が現れます。<br>


## Requirement
| name | version |
| ------------- | ------------- |
| nuxt  | 2.x |
| @nuxtjs/composition-api  | ^0.31.0 |
| firebase | ^9.6.6 |

## Getting Started
0. add `.env` file

1. `yarn`<br>

2. `yarn dev`<br>

3. Access to [Localhost:3000](http://localhost:3000/)

## Deploy Service
